project starfall

Made with unity en C#

The github is private

OperationStarfall is a hybrid learning environment created by Neon Gamestudio In a hybrid learning environment, you learn by working on a game for the game market in a professional environment.

We are working on a couch co-op metroidvania with a team of artist and developers. The first 10-week period I was there was all about gameplay. The period was split into two sprints. The first two sprint I worked with someone else to create a laser turret for in the levels. In the second peroid I worked with someone else on a custom editor script for unity which makes it easy to add object in a chain and you can easily and specifically place new objects in the same chain and that they will be placed in the right place in a unity diconery and in the scene
