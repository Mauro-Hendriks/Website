about me

Ik ben Mauro Hendriks uit Purmerend ik doe op dit moment een opleiding software develpment op het Media College in Amsterdam.

In De afgelopen twee jaar heb ik op school meerdere projecten gemaakt met unity en C# hierbij heb ik veel verschillende Dingen zoals combat, movement, development tools en met 2d assets een walkable environment maken. Ik heb wat verschillende dingen geprobeerd om te kijken of ik iets kan vinden waar in ik zou willen specialiseren op dit moment heb ik dat nog niet gevonden dus hopelijk kan ik het op stage wel vinden want dan kan ik iets voor focussen op het leren van dat onderdeel. Zelf speel ik ook veel game zoals Monster Hunter Rise, Slime Rancher,cyberpunk 2077, Overwatch 2 en Outer wilds
